LOCATION 0 Entrance Hall
DESC A grand entrance hall draped in cobwebs. A faded portrait watches you from the far wall. Dust motes hang in the air.
ATMO The entrance hall. The portrait still watches.
EXIT north 1 0 0
EXIT east 2 0 0
END

DESC A long corridor with peeling wallpaper. Doors line both sides, all locked except the one at the far end.

END

END
