# Dialogue scripts for Project Yozora
# Format: NODE <id> <speaker>
#         TEXT <dialogue text>
#         CHOICE <id> <next_node> <text>
#         TERMINAL
#         END
#
# Dialogue trees are built procedurally in dialogue.c
# (dialogue_build_for_location). This file documents the intended scripts.

# ── Entrance Hall (location 0) ─────────────────────────────────────────
DIALOGUE entrance_hall
NODE 0 You
TEXT The entrance hall stretches before me. Dust motes float in the sickly
     light. I shouldn't be here.
CHOICE 0 1 ...
END
NODE 1 You
TEXT The front door is swollen shut with moisture. I have no choice but to
     go deeper.
TERMINAL
END

# ── Dark Corridor (location 1) ─────────────────────────────────────────
DIALOGUE dark_corridor_locked
NODE 0 You
TEXT The basement door is heavy iron, cold as winter. A padlock the size of
     my fist hangs on the latch.
CHOICE 0 1 ...
END
NODE 1 You
TEXT I need a key. Maybe there is one somewhere in this house.
TERMINAL
END

DIALOGUE dark_corridor_unlocked
NODE 0 You
TEXT The brass key fits perfectly. The padlock clicks open with a sound like
     a gunshot in the silence.
CHOICE 0 1 ...
END
NODE 1 You
TEXT Cold air rushes up from the darkness below. Whatever is down there has
     been waiting a long time.
TERMINAL
END

# ── The Library (location 2) ───────────────────────────────────────────
DIALOGUE library_diary
NODE 0 You
TEXT A leather-bound diary lies open on the desk. The ink is faded but
     still legible.
CHOICE 0 1 ...
END
NODE 1 Professor Ashwood's Diary
TEXT October 12th. The ritual is complete. The creature is bound. But the
     price... God forgive me, the price was Lily.
CHOICE 0 2 ...
END
NODE 2 Professor Ashwood's Diary
TEXT It will keep for a generation. The circle holds. But someone will come
     eventually. God help them.
TERMINAL
END

# ── Child's Room (location 4) ──────────────────────────────────────────
DIALOGUE childs_room_key
NODE 0 You
TEXT A small brass key sits in the toy box, half-buried under a threadbare
     teddy bear.
CHOICE 0 1 ...
END
NODE 1 Lily
TEXT That is for the door downstairs.
CHOICE 0 2 ...
END
NODE 2 Lily
TEXT Do not open it. Please. Whatever happens... do not open that door.
TERMINAL
END

# ── Ritual Room (location 5) ───────────────────────────────────────────
DIALOGUE ritual_room
NODE 0 You
TEXT The symbols carved into the floor pulse with a dull red light. The air
     tastes of iron and old fear.
CHOICE 0 1 ...
END
NODE 1 You
TEXT I can see the shape of the ritual now. A binding. Something was
     imprisoned here. Something that has been slowly... waking up.
CHOICE 0 2 ...
END
NODE 2 Lily
TEXT You found the truth. There is a name written in the centre. Speak it
     aloud and it ends. But only if you are sure. Only if you are brave.
TERMINAL
END
